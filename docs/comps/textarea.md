# TextArea多文本框

## 基本用法
::: demo
```html
<template>
  <textarea class="hi-textarea" placeholder="请输入..."  width='100%' rows="5"></textarea>
</template>
<style>
  @import '../packages/styles/textarea.css'
</style>
```
:::


## 禁用状态

<template>
